# Side Panel 主界面 UI 设计文档

## 页面模块

1. Header 顶部状态区
2. 当前图片区域
3. 提示词结果区域
4. 图像生成设置区域
5. 生成结果区域
6. 底部任务状态区

## 布局

```text
┌──────────────────────────────────────┐
│ PromptPilot                           │
│ [Prompt API: 正常] [Image API: 未配置] │
├──────────────────────────────────────┤
│ 当前图片                              │
│ [图片预览]                            │
│ [反推] [清空]                         │
├──────────────────────────────────────┤
│ 提示词结果                            │
│ tags                                  │
│ 中文提示词 textarea                    │
│ English Prompt textarea               │
│ [复制中文] [复制英文] [优化中文] [优化英文] │
├──────────────────────────────────────┤
│ 图像生成                              │
│ Provider / Model / Mode / Size         │
│ [生成图片] [混合生成4张]                │
├──────────────────────────────────────┤
│ 生成结果 2 x 2                         │
└──────────────────────────────────────┘
```

## 设计风格

- 暗色科技感
- 圆角卡片
- 青绿色主按钮
- 状态徽章明确区分接口状态
